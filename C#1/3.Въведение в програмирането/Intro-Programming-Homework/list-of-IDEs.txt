1.Visual Studio
Visual Studio is the mostly used by corporate programmers to produce .NWT code.
Advantages of this IDE are Intellisense, Automatic code formatting,Extensibility.

2.Eclipse
This is the best alternative for those who are not a big fans of Microsoft'ss Visual Studio.
The interface is a lot cleaner and concise than Visual Studio and the languages that are possible
to use extend past the .NET Framework. The advantages of this IDE are Language Extensibility, Free,Steep Learning Curve.

3.Netbeans
Netbeans is number one alternative to Eclipse. This program is extensible and free, and it compiles code which is a bonus.
When installing Netbeans it comes with a Java server called GlassFish that you could deploy on your computer for your testing
environment. Advantages of this IDE are Extensible, Compiles Code, Free, Bloated.

4.Kate
Kate is exactly how the website describes it, a text editor. Not to be looked down upon as it does a lot more than a regular
text edit can do and that is what makes this an indispensable product.
The Advantages of this IDE are Uses Linux, Code Floding, KDE Overhead, Doesen''t Compile Code

5.gedit

gedit is the official editor of the gnome desktop and comes installed once the system is setup.
Very useful for the user that is casually editing
code and not undertaking major projects. This would be a good companion to Netbeans or Eclipse.
The Advantages of this IDE are Uses Linux, Already Installed on gnome systems,
Code Highlighting, Tabbed, Doesn''t Compile Code
